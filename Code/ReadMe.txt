Steps to run "MarkovClustering.py"

1) Place the dataset along with the .py file in the same directory.
2) Run MarkovClustering.py
3) Input the 3 parameters as asked by the program. a) dataset file name b) expansion parameter c) Inflation parameter
4) .clu file is generated in the same directory of the .py file.
5) Use the .clu file in pajeck for cluster visualization.
